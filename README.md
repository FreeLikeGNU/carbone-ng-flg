=======================
This is a modded fork of the Carbone NG game for Minetest
https://forum.minetest.net/viewtopic.php?f=15&t=12824

To use this game with Minetest, insert this repository as
  /games/carbone-ng-flg-modded
in the Minetest Engine.
You must use Minetest at least the latest 0.4.12 build from github or greater
see the documentation on how to build here: https://github.com/minetest/minetest

then:

$ git clone -b modded https://github.com/FreeLikeGNU/carbone-ng-flg.git --recursive carbone-ng-flg-modded


The Minetest Engine can be found in:
  https://github.com/minetest/minetest/
